## Project Overview

**The Wedding Post** — A wedding RSVP site for Iara Mello & Douglas Tertuliano (wedding: 21/11/2026). Guests find their family group, confirm attendance for each member, and the data goes to Google Sheets via Google Apps Script. No backend or database — just Next.js + Google Sheets.

## Architecture

```
app/              # Next.js App Router (no src/ directory)
data/guests.md    # Guest list — source of truth for all groups/members
lib/guests.ts     # Parser for guests.md + Fuse.js search
lib/sheets.ts     # Google Apps Script client (GET check + POST submit)
lib/storage.ts    # localStorage helpers for confirmed-group tracking
public/           # Static assets
```

### Data Flow

1. `data/guests.md` is read server-side by `lib/guests.ts` → parsed into `GuestGroup[]`
2. The guest array is passed as a prop to the client-side search component (Fuse.js, threshold 0.35)
3. On confirmation: check `localStorage` first → GET to Apps Script to check for duplicates → POST to save
4. Apps Script writes one row per family member to the `rsvp` tab in Google Sheets

### Apps Script Integration Notes

- POST requests to Apps Script **must** use `Content-Type: text/plain` (not `application/json`) — CORS requirement
- GET: `?action=check&groupId=<slug>` → `{ confirmed: boolean }`
- POST body: `RSVPPayload` as JSON string
- `groupId` is a slug derived from the family name: "Família Silva" → `"familia-silva"`

## Design System

The visual concept is a vintage printed newspaper ("The Wedding Post"). All design decisions must reinforce this.

**Fonts (Google Fonts):** Playfair Display (headings, 400/700/900) + IM Fell English (body text). **Never use sans-serif fonts.**

**Color palette:**
- Background: `#f5f0e8` (aged cream paper)
- Text: `#1a1a1a` (typographic black)
- Accent/CTA: `#c4683a` (terracotta) — buttons, borders, decorative lines
- Muted text: `#8c7355`
- Borders: `#d4c9b0`

**UI rules:**
- `border-radius` max 4px — sharp corners everywhere
- No vibrant colors, no gradients, no large shadows
- No emojis (breaks the editorial style)
- Double-rule dividers (`3px solid` + `1px solid` ink, 2px apart) between sections
- shadcn/ui components are used but overridden with serif fonts, radius 0, and terracota focus states

## guests.md Format

```markdown
## Família Silva
- João Silva
- Maria Silva

## Rodrigo Almeida
- Rodrigo Almeida
```

Groups are `## H2` headings; members are `- ` list items. The slug/`groupId` is auto-generated by stripping accents and replacing spaces with hyphens.
